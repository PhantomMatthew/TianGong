IMPLEMENTATION DETAILS:
======================
1. Agent struct:
   - provider: LLM provider interface
   - tools: Tool registry
   - store: SessionStore interface
   - config: AgentConfig (MaxIterations, HistoryLimit, SystemPrompt)

2. New() constructor:
   - Applies default MaxIterations=10
   - Applies default HistoryLimit=50
   - Returns configured Agent

3. RunStream() ReAct loop:
   - Adds user message to session
   - Iterates with max iterations guard (default 10)
   - Each iteration:
     * Gets message history
     * Applies history limit (keeps last N messages)
     * Builds system prompt with tools via FormatSystemPrompt()
     * Builds provider messages via BuildMessages()
     * Calls provider.ChatStream()
     * Streams text deltas to writer
     * If finish_reason=tool_calls:
       - Executes tools sequentially (no parallel)
       - Adds tool results to session
       - Continues loop
     * If finish_reason=stop:
       - Adds assistant message
       - Returns success
     * If finish_reason=max_tokens:
       - Treats as completion
       - Returns success
   - If max iterations reached: returns error

4. executeTool() helper:
   - Looks up tool in registry
   - Returns error if unknown tool
   - Executes tool with JSON arguments
   - Returns result or error

5. Error Handling:
   - Provider errors: propagated immediately
   - Tool execution errors: converted to error message as tool result
   - Unknown tools: error message added as tool result
   - Max iterations: descriptive error returned
   - Stream errors: propagated immediately

6. Logging (log/slog):
   - Iteration start (iteration #, session ID)
   - Stream finish (finish_reason, tool_calls count)
   - Tool execution (tool name, result length)
   - Tool failures (tool name, error)
   - Agent completion (iterations count)

7. Tests (6 scenarios):
   - TestNew: default and custom config
   - TestRunStream_SimpleChat: basic conversation without tools
   - TestRunStream_ToolCall: single tool call execution
   - TestRunStream_MaxIterations: iteration guard enforcement
   - TestRunStream_UnknownTool: unknown tool error handling
   - TestRunStream_HistoryLimit: history truncation
